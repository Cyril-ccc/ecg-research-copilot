# Report Template

## Required sections
1. Cohort definition
2. Data & QC
3. Results
4. Limitations

## Section guidance

### Cohort definition
- Question summary
- Template name and parameters
- Inclusion/exclusion notes

### Data & QC
- Data sources and artifacts
- QC pass rate and fail reasons summary
- Missingness overview

### Results
- Core feature summaries
- Group comparisons
- Key plots and table references

### Limitations
- Missing data
- Confounding
- Time-window sensitivity
- External validity limits

## Traceability
- Every claim should map to artifacts/tables.
- Mention relevant artifact filenames in the text.
